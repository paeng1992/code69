#### Tested on:
- IPhone (**ish**).
- Android (**Termux**).
- Windows
- Linux

## Installation
### Android (Termux):
- `pkg install git`
- `pkg install python python-pip`
- `git clone https://github.com/paeng1992/code69.git`
- `cd code69`
- `pip install -r requirements.txt`
- `python main.py`

### IPhone (ISH):
- `apk add git`
- `apk add python3`
- `apk add py3-pip`
- `git clone https://github.com/paeng1992/code69.git`
- `cd code69`
- `pip install -r requirements.txt`
- `python3 main.py`
